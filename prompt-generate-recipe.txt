Génère une commande complète pour créer une recette anti-gaspillage.

CONTRAINTES STRICTES :
- La recette DOIT être une vraie recette anti-gaspillage (utiliser restes, épluchures, produits abîmés, etc.)
- Réponds UNIQUEMENT avec la commande complète, sans texte avant ou après
- Pas d'explications, pas de markdown, pas de commentaires

FORMAT OBLIGATOIRE :
echo 'JSON' | docker compose exec -T laravel.test php artisan recipe:create lulu.bruyere43@gmail.com

STRUCTURE JSON REQUISE (compact, sans retours à la ligne) :
{"title":"Titre","summary":"Description","servings":4,"prep_minutes":10,"cook_minutes":20,"difficulty":"easy","cuisine":"Française","calories":200,"is_public":true,"ingredients":[{"name":"Nom","quantity":100,"unit":"g","position":1}],"steps":[{"text":"Description","position":1}],"image_url":"https://url.jpg"}

CHAMPS OBLIGATOIRES :
- title : Titre court et accrocheur
- summary : Description détaillée (2-3 phrases)
- servings : Nombre de portions (entier)
- difficulty : "easy", "medium" ou "hard"
- ingredients : Array avec minimum 3 ingrédients
- steps : Array avec minimum 3 étapes

CHAMPS OPTIONNELS :
- prep_minutes : Temps de préparation en minutes
- cook_minutes : Temps de cuisson en minutes
- cuisine : Type de cuisine (Française, Italienne, etc.)
- calories : Calories par portion
- is_public : true/false (défaut: true)
- image_url : URL image (si absent, image par défaut utilisée)

INGRÉDIENTS :
{
  "name": "Nom de l'ingrédient",
  "quantity": nombre (peut être décimal),
  "unit": "g|kg|ml|l|piece|slice|tbsp|tsp|clove|branch|leaf|jar|sachet|roll|pinch|grain",
  "position": numéro ordre (commence à 1)
}

ÉTAPES :
{
  "text": "Description de l'étape",
  "position": numéro ordre (commence à 1),
  "timer_minutes": durée optionnelle
}

DIFFICULTÉS :
- easy : recettes simples, techniques basiques
- medium : techniques moyennes
- hard : techniques avancées

THÈMES ANTI-GASPI :
- Pain rassis (pain perdu, chapelure, croûtons)
- Fruits abîmés (compotes, smoothies, confitures)
- Épluchures (chips, bouillons, pestos)
- Fanes de légumes (soupes, tartes, gratins)
- Restes (croquettes, hachis, gratins)
- Yaourts périmés (gâteaux, sauces)
- Fromages secs (gratins, sauces)
- Légumes flétris (soupes, purées)

RÈGLES JSON :
- JSON sur UNE SEULE LIGNE (pas de retours à la ligne)
- Pas d'espaces inutiles
- Utiliser guillemets doubles pour les clés et valeurs
- Échapper les apostrophes dans le texte : remplacer ' par \'
- Échapper les guillemets doubles dans le texte : \"
- Ne PAS mettre de retour à la ligne dans la commande
