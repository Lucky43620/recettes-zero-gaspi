Génère une commande complète pour créer une recette anti-gaspillage dans une application Laravel.

CONTRAINTES STRICTES :
- La recette DOIT être une vraie recette anti-gaspillage (utiliser restes, épluchures, produits abîmés, etc.)
- Réponds UNIQUEMENT avec la commande Docker, sans aucun texte avant ou après
- Pas d'explications, pas de markdown, pas de commentaires
- Format : docker compose exec -T laravel.test php artisan recipe:create lulu.bruyere43@gmail.com 'JSON'

STRUCTURE JSON REQUISE :
{
  "title": "Titre court et accrocheur",
  "summary": "Description détaillée de la recette (2-3 phrases)",
  "servings": nombre_de_portions,
  "prep_minutes": temps_préparation,
  "cook_minutes": temps_cuisson,
  "difficulty": "easy|medium|hard",
  "cuisine": "Type de cuisine",
  "calories": calories_par_portion,
  "is_public": true,
  "ingredients": [
    {
      "name": "Nom ingrédient",
      "quantity": quantité_numérique,
      "unit": "g|kg|ml|l|piece|slice|tbsp|tsp|clove|branch|leaf|jar|sachet|roll|pinch|grain",
      "position": position_ordre
    }
  ],
  "steps": [
    {
      "text": "Description détaillée de l'étape",
      "position": position_ordre,
      "timer_minutes": durée_optionnelle
    }
  ],
  "image_url": "https://url-image-libre-droits.jpg"
}

RÈGLES D'INGRÉDIENTS :
- Minimum 3 ingrédients, maximum 15
- Utiliser des ingrédients qui sont normalement jetés (épluchures, pain rassis, fruits abîmés, restes, etc.)
- Units valides uniquement : g, kg, ml, l, piece, slice, tbsp, tsp, clove, branch, leaf, jar, sachet, roll, pinch, grain
- Position commence à 1 et s'incrémente

RÈGLES D'ÉTAPES :
- Minimum 3 étapes, maximum 12
- Chaque étape doit être claire et précise
- Position commence à 1 et s'incrémente
- timer_minutes uniquement si nécessaire (cuisson, repos, etc.)

RÈGLES D'IMAGE :
- URL doit être accessible publiquement
- Préférer des images libres de droits (Unsplash, Pexels, etc.)
- Format : jpg, jpeg, png, webp

DIFFICULTÉS :
- easy : recettes simples, peu d'étapes, techniques basiques
- medium : techniques moyennes, plusieurs étapes
- hard : techniques avancées, préparation complexe

TYPES DE CUISINE :
Française, Italienne, Asiatique, Méditerranéenne, Végétarienne, Végane, Sans gluten, etc.

EXEMPLE DE THÈMES ANTI-GASPI :
- Pain rassis (pain perdu, chapelure, croûtons)
- Fruits abîmés (compotes, smoothies, confitures)
- Épluchures (chips, bouillons, pestos)
- Fanes de légumes (soupes, tartes, gratins)
- Restes de viande/poisson (croquettes, hachis, gratins)
- Yaourts périmés (gâteaux, sauces)
- Fromages secs (gratins, sauces)
- Légumes flétris (soupes, purées, gratins)

ÉCHAPPEMENT JSON :
- Échapper les guillemets doubles dans le JSON avec \"
- Échapper les apostrophes avec \'
- Le JSON complet doit être entouré de guillemets simples dans la commande

IMPORTANT :
- Une seule ligne de commande
- Pas de retour à la ligne dans la commande
- JSON compact (pas de pretty print)
- Vérifier que le JSON est valide avant de générer la commande
