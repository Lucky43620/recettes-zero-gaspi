Génère une commande complète pour créer une recette culinaire.

ÉTAPE 1 - GÉNÉRATION D'IMAGE (si disponible) :
Si tu as accès à DALL-E ou un outil de génération d'images :
1. Génère d'abord une image appétissante pour la recette
2. Récupère l'URL de l'image générée
3. Utilise cette URL dans le champ "image_url" du JSON

Si tu n'as pas accès à la génération d'images :
- Omets complètement le champ "image_url" (une image par défaut sera utilisée)

ÉTAPE 2 - GÉNÉRATION DE LA COMMANDE :

CONTRAINTES :
- Réponds UNIQUEMENT avec la commande, sans texte avant ou après
- Pas d'explications, pas de markdown, pas de commentaires
- Commande sur UNE SEULE LIGNE, sans retour à la ligne

FORMAT OBLIGATOIRE :
echo 'JSON_COMPACT' | docker compose exec -T laravel.test php artisan recipe:create lulu.bruyere43@gmail.com

STRUCTURE JSON (tout sur une ligne, sans espaces inutiles) :
{"title":"...","summary":"...","servings":4,"prep_minutes":10,"cook_minutes":20,"difficulty":"easy","cuisine":"...","calories":200,"ingredients":[...],"steps":[...]}

CHAMPS OBLIGATOIRES :
- title : Titre de la recette (court et accrocheur)
- summary : Description détaillée (2-3 phrases)
- servings : Nombre de portions (entier, ex: 4)
- difficulty : UNIQUEMENT "easy", "medium" ou "hard"
- ingredients : Array avec minimum 3 ingrédients
- steps : Array avec minimum 3 étapes

CHAMPS OPTIONNELS :
- prep_minutes : Temps de préparation en minutes (entier)
- cook_minutes : Temps de cuisson en minutes (entier)
- cuisine : Type de cuisine (Française, Italienne, Asiatique, etc.)
- calories : Calories par portion (entier)
- image_url : URL image (si omis, image par défaut utilisée automatiquement)

STRUCTURE INGRÉDIENT (obligatoire) :
{
  "name": "Nom de l'ingrédient",
  "quantity": nombre_décimal,
  "unit": "CODE_UNITÉ",
  "position": numéro_ordre
}

UNITÉS DISPONIBLES (utiliser UNIQUEMENT ces codes) :
- POIDS : "g" (grammes), "kg" (kilogrammes), "mg" (milligrammes)
- VOLUME : "ml" (millilitres), "l" (litres), "cl" (centilitres), "dl" (décilitres)
- MESURES : "tsp" (cuillère à café), "tbsp" (cuillère à soupe), "cup" (tasse)
- UNITÉS : "piece" (pièce), "slice" (tranche), "pinch" (pincée), "bunch" (botte), "can" (boîte), "jar" (pot), "package" (paquet), "branch" (branche), "leaf" (feuille), "clove" (gousse), "sachet" (sachet), "roll" (rouleau), "grain" (grain)

STRUCTURE ÉTAPE (obligatoire) :
{
  "text": "Description claire de l'étape",
  "position": numéro_ordre,
  "timer_minutes": durée_optionnelle
}

NIVEAUX DE DIFFICULTÉ :
- "easy" : recettes simples, techniques basiques, peu d'étapes
- "medium" : techniques moyennes, plusieurs étapes
- "hard" : techniques avancées, préparation complexe

RÈGLES IMPORTANTES :
1. JSON sur UNE SEULE LIGNE (aucun retour à la ligne dans le JSON)
2. Pas de retour à la ligne dans la commande echo
3. Utiliser guillemets doubles " pour toutes les valeurs textuelles JSON
4. ÉCHAPPEMENT DES APOSTROPHES (ULTRA IMPORTANT - NE PAS SE TROMPER) :

   RÈGLE D'OR : Chaque apostrophe ' dans le texte français doit être remplacée par EXACTEMENT cette séquence de 5 caractères : '\''

   Détail de la séquence '\'' :
   - Caractère 1 : ' (apostrophe qui ferme la chaîne)
   - Caractère 2 : \ (backslash d'échappement)
   - Caractère 3 : ' (apostrophe échappée)
   - Caractère 4 : ' (apostrophe qui ouvre la chaîne)
   - Total : '\'' (5 caractères)

   ⚠️ ATTENTION : NE JAMAIS utiliser ''' (3 apostrophes) - C'EST FAUX !
   ✓ TOUJOURS utiliser '\'' (apostrophe-backslash-apostrophe-apostrophe)

   Exemples de transformation :
   ✓ CORRECT : d'eau → d'\''eau
   ✓ CORRECT : l'ail → l'\''ail
   ✓ CORRECT : jusqu'à → jusqu'\''à
   ✗ FAUX : d'''eau (3 apostrophes - NE JAMAIS FAIRE ÇA)
   ✗ FAUX : d eau (sans apostrophe - pas français)

5. Positions des ingrédients/étapes commencent à 1 et s'incrémentent
6. La recette sera publique automatiquement
7. Ne PAS ajouter le champ "is_public" (toujours true par défaut)
8. Les images générées avec DALL-E sont automatiquement téléchargées et sauvegardées sur le serveur

TABLEAU DE CONVERSION DES APOSTROPHES (À SUIVRE EXACTEMENT) :
┌──────────────────────┬─────────────────────────┐
│ Texte français       │ Dans le JSON            │
├──────────────────────┼─────────────────────────┤
│ d'eau                │ d'\''eau                │
│ l'ail                │ l'\''ail                │
│ l'huile              │ l'\''huile              │
│ l'oignon             │ l'\''oignon             │
│ jusqu'à              │ jusqu'\''à              │
│ qu'il                │ qu'\''il                │
│ qu'elle              │ qu'\''elle              │
│ s'il                 │ s'\''il                 │
└──────────────────────┴─────────────────────────┘
